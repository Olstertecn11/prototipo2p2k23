Formulario de Examen


Funcionamiento:

Boton Cargar Datos

Este boton al darle click cargara las tablas de la base de datos dentro del combobox, posteriormente podra seleccionar una tabla en el combobox

Boton Ayuda

Este boton mostrara un pequeño forulario de ayuda, describiendo el funcionamiento del formulario

Boton Reporte

Este boton abrira una pequeña ventana de Crystal Reports donde podremos visualizar los datos de la tabla seleccionada

Boton Salir

En este boton saldremos de nuestra aplicacion :D
